# 공터 일조권 시뮬레이터

주변 건물의 그림자가 특정 공터에 미치는 영향을 시뮬레이션하는 인터랙티브 웹 앱입니다.

## 기능

- 2D 평면도
  - 건물 배치 및 그림자를 캔버스로 렌더링
  - 건물 드래그 / 추가 / 삭제 / 높이 조절
- 3D 뷰
  - Three.js 기반 3D 씬
  - 마우스로 회전 / 줌 가능
- 일조 그래프
  - Chart.js 이중 Y축 차트
  - 시간대별 일조 면적(%) 및 태양 고도각 표시
- 하루 재생
  - 시간 슬라이더 애니메이션으로 그림자 변화 확인
- 도시 선택
  - 서울 / 부산 / 제주 / 도쿄 / 뉴욕 위도 프리셋

## 사용 기술

- React 19
  - Hooks: `useState`, `useEffect`, `useRef`, `useMemo`
- Three.js
  - 3D 렌더링
  - PCF Soft Shadow Map
- Chart.js
  - 반응형 바/라인 복합 차트
- Canvas 2D API
  - 커스텀 2D 렌더러

## 핵심 알고리즘

- 태양 위치 계산
  - 위도 및 Day of Year 기반 고도각 / 방위각 천문 공식
- Convex Hull
  - Andrew's Monotone Chain으로 그림자 다각형 생성
- Point-in-Polygon
  - Ray Casting으로 일조 면적 샘플링 계산

## 실행 방법

```bash
npm install
npm run dev
```

## Phase 1 — 실사용자 연결

가장 중요한 단계. 이 단계가 들어가야 데모가 아니라 실제 서비스가 된다.

### 1-1. 실제 주소 검색 + 지도 연동

- 카카오맵 API 연동
  - 주소 검색 시 해당 위치로 즉시 이동
- 선택한 위치의 위도 / 경도 자동 반영
- 예시
  - `"서울 강남구 역삼동 123"` 입력
  - 바로 해당 위치 기준으로 시뮬레이션 시작
- 목표
  - 사용자가 임의 좌표를 수동 입력하지 않아도 바로 체험 가능해야 함

### 1-2. 공공데이터 건물 높이 자동 로딩

- 국토부 건물통합정보 API 연동
  - 건물 높이
  - 층수
  - 용도
- 또는 VWORLD 3D 건물데이터 API 연동
- 주소 입력 시 동작
  - 반경 200m 내 건물 자동 배치
  - 각 건물 높이 자동 반영
- 수동 건물 추가 기능
  - 유지
  - 용도: 신축 시뮬레이션, 가상 건물 배치 테스트

### 1-3. Vercel / Netlify 배포

- 실서비스 URL 확보
  - 예: `solar-shadow.vercel.app`
- 포트폴리오용 라이브 링크 제공
  - 면접관이 설치 없이 즉시 체험 가능
- 필수 배포 마감 항목
  - OG 메타태그 설정
  - 프리뷰 이미지 설정
